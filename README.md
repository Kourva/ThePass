# ThePass
8 Length Password Generator Android app sing Kivy

# Install (If you want to run it on Linux or Windows)
+ clone
```bash
git clone https://github.com/Kourva/ThePass && cd ThePass && chmod +x install.sh
```
+ requirements
```bash
./install.sh
```
+ run
```bash
python3 main.py
```
